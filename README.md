Huffman Encoding using Java

A Java implementation of the Huffman Coding algorithm for lossless data compression, built from scratch without relying on Java's built-in PriorityQueue. The project constructs a Huffman Tree using a custom Generic Min Heap and performs efficient text encoding and decoding.

Features
Lossless text compression using Huffman Coding
Custom Generic Min Heap implementation
Huffman Tree construction using Greedy Algorithm
Fast encoding and decoding with HashMaps
Object-Oriented and modular Java design
Time Complexity: O(n log n) for Huffman Tree construction
